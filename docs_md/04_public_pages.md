# 04 — Публичные страницы

## Главная страница

**Файл:** `index.php`

### Структура

1. **Подключение ядра:**
   - `includes/config.php` — подключение к БД, настройки сессии
   - `includes/functions.php` — функции покупателя

2. **Загрузка данных:**
   - `getProducts(6)` — 6 последних товаров
   - `getLatestReviews(5)` — 5 последних отзывов

3. **Секции страницы:**

   #### Hero-секция

   - Приветственный блок с заголовком "Волшебная ЛАВКА"
   - Кнопка перехода в каталог ("Перейти к покупкам")
   - Фоновое изображение из `images/background/fon.jpg`

   #### Новые товары

   - Сетка из 3 карточек товаров (`.products-grid`)
   - Каждая карточка: изображение, название, цена, кнопка "В корзину"
   - Кнопки "В корзину" работают через AJAX (`js/script.js`)

   #### Последние отзывы

   - Блок с 3 последними отзывами
   - Каждый отзыв: текст, автор, дата, название товара
   - Ссылка на товар

4. **Подключение:**
   - `includes/header.php` — шапка сайта
   - `includes/footer.php` — подвал сайта
   - `js/script.js` — инициализация корзины

## Каталог товаров

**Файл:** `shop.php`

### Режимы работы

#### Режим 1: Список всех товаров (без GET параметра)

1. **Загрузка данных:**
   - `getProducts()` — все товары без лимита

2. **Отображение:**
   - Заголовок "Каталог товаров"
   - Сетка карточек (`.products-grid`)
   - Каждая карточка: изображение, название, описание (обрезанное), цена, кнопка "В корзину"

3. **Переход к детальной странице:**
   - Клик по карточке → `shop.php?id=productId`

#### Режим 2: Детальная страница товара (GET `?id=`)

1. **Загрузка данных:**
   - `getProduct($productId)` — информация о товаре
   - `getProductReviews($productId)` — все отзывы на товар

2. **Отображение:**
   - Крупное изображение товара
   - Название, описание, цена
   - Кнопка "В корзину"
   - Блок отзывов (если есть):
     - Текст отзыва, автор, дата
   - Форма добавления отзыва (если пользователь авторизован и `canReviewProduct()` = true)

3. **Логика отзыва:**
   - Проверка: купил ли пользователь товар (completed заказ)
   - Проверка: не оставлял ли он уже отзыв
   - Если обе проверки пройдены — показ формы
   - POST формы → `addReview()` → редирект на ту же страницу

### Зависимости

- `includes/functions.php` — товары, отзывы
- `includes/header.php` — шапка (с кнопкой корзины)
- `includes/footer.php` — подвал
- `js/script.js` — AJAX корзина

## Страница входа

**Файл:** `login.php`

### Структура страниц

1. **Подключение:**
   - `includes/config.php`
   - `includes/functions.php`

2. **Обработка POST:**
   - Получение `login` (email или username) и `password`
   - Вызов `universalLogin($login, $password)`
   - При успехе — редирект по роли
   - При ошибке — установка `$error` для отображения

3. **Форма:**
   - Поле "Email или логин"
   - Поле "Пароль"
   - Кнопка "Войти"
   - Ссылка на регистрацию: `users/register.php`
   - Отображение ошибок через `$error`

4. **Стилизация:**
   - Прямое подключение `css/style.css` (без header.php)
   - Центрированная карточка формы

## Общие компоненты

### Шапка сайта

**Файл:** `includes/header.php`

**Содержимое:**

- Логотип "Волшебная ЛАВКА" (ссылка на `/`)
- Навигация:
  - "Каталог" → `/shop.php`
  - "Корзина" → `/cart/cart.php` (с бейджем количества)
- Блок пользователя (если авторизован):
  - Имя пользователя
  - Выпадающее меню:
    - "Профиль" → `/users/profile.php`
    - "Админ-панель" → `/admin/index.php` (только admin)
    - "Модератор" → `/moderator/index_md.php` (только moderator)
    - "Выйти" → соответствующий logout.php
- Кнопка "Войти" / "Регистрация" (если не авторизован)

**Стилизация:**

- Фиксированная шапка (position: sticky)
- CSS Grid/Flexbox
- Адаптивность (@media 768px)

### Подвал сайта

**Файл:** `includes/footer.php`

**Содержимое:**

- Копирайт "© 2025 Волшебная ЛАВКА"
- Указание команды: "Разработано командой АВВА"
- Текущий год через `date('Y')`

## Поток данных на публичных страницах

```php
index.php
  ├── getProducts(3) ──────────────────→ products (3 последних)
  ├── getLatestReviews(3) ───────────→ reviews (3 последних с авторами)
  ├── header.php ────────────────────→ навигация, корззина, пользователь
  ├── HTML (hero, products, reviews) ─→ рендер на странице
  ├── script.js ─────────────────────→ initCart(), AJAX обработчики
  └── footer.php ────────────────────→ копирайт

shop.php
  ├── GET ?id= ──────────────────────→ режим детальной страницы
  │   ├── getProduct(id) ────────────→ товар
  │   ├── getProductReviews(id) ───→ отзывы
  │   ├── canReviewProduct() ──────→ можно ли оставить отзыв
  │   └── POST отзыв ────────────────→ addReview()
  │
  └── Без GET ───────────────────────→ режим каталога
      ├── getProducts() ─────────────→ все товары
      ├── header.php
      ├── HTML (product grid)
      ├── script.js
      └── footer.php

login.php
  ├── POST login/password ──────────→ universalLogin()
  │   ├── Поиск по username/email
  │   ├── password_verify()
  │   └── Редирект по роли
  │
  └── GET ───────────────────────────→ форма входа
```
