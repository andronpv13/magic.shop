[![PHP Version](https://img.shields.io/badge/php-7.0%2B-blue.svg)](https://php.net)

## Технологический стек

- Backend: PHP 7+ (mysqli, prepared statements)
- Database: MySQL / MariaDB (InnoDB, utf8mb4)
- Frontend: HTML5, CSS3 (CSS Variables, Grid, Flexbox), Vanilla JavaScript (Fetch API)
- Authentication: PHP Sessions + bcrypt

## О проекте

**MagicShop** — полнофункциональный интернет-магазин с многоуровневой системой ролей (admin / moderator / customer), корзиной покупок, оформлением заказов, оплатой и системой отзывов.

**Основное правило:**

- Не вносить изменения в документацию, если они не были согласованы с авторами проекта.

## Быстрый старт

### 1. Создание базы данных

```sql
--- Удаление базы данных если существует ---
DROP DATABASE IF EXISTS shop_db;
--- Создание базы данных ---
CREATE DATABASE shop_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. Импорт схемы

```bash
mysql -u root -p shop_db < database/schema.sql
```

### 3. Настройка подключения

Отредактировать `includes/config.php`:

```php
$host = 'localhost';
$db_name = 'shop_db';
$db_user = 'root';
$db_pass = 'toor';
```

### 4. Запуск

Открыть в браузере: `http://localhost`

### 5. Вход администратора

- URL: `http://localhost/login.php`
- Логин: `root`
- Пароль: `toor` ваш пароль для дашборда, желательно сменить

## Схема базы данных

### Таблицы

| Таблица | Описание |
|---|---|
| **users** | Пользователи (admin, moderator, customer) |
| **products** | Товары с привязкой к создателю |
| **orders** | Заказы покупателей |
| **order_items** | Позиции заказов |
| **reviews** | Отзывы покупателей |

### Статусы заказов

| Статус | Описание | Цвет |
|---|---|---|
| `pending` | Ожидает обработки | жёлтый |
| `payment` | Ожидает оплаты | синий |
| `completed` | Оплачен | зелёный |
| `cancelled` | Отменён | красный |

## AJAX API (корзина)

| Метод | URL | Параметры | Ответ |
|---|---|---|---|
| POST | `/cart/add_to_cart.php` | product_id, quantity | {success, message, cart_count} |
| POST | `/cart/remove_from_cart.php` | product_id | {success, message, cart_count, cart_total} |
| POST | `/cart/update_cart.php` | product_id, quantity | {success, message, cart_count, item_total, cart_total} |

## Безопасность

| Мера | Реализация |
|---|---|
| CSRF-защита | Во всех страницах |
| SQL-инъекции | Prepared statements (mysqli) |
| XSS | htmlspecialchars(ENT_QUOTES, 'UTF-8') |
| Пароли | bcrypt (password_hash) |
| Session hijacking | httponly=true, samesite=Strict |
| Session timeout | 10 мин неактивности |
| Session fixation | Регенерация ID каждые 30 мин |
| File upload | Проверка расширения, лимит 5MB |
