Для работы с проектом "magic.shop" рекомендуется использовать следующие команды:

1. **Настройка базы данных**:
   - Создание базы данных: `mysql -u root -p -e "CREATE DATABASE shop_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"`
   - Импорт схемы: `mysql -u root -p shop_db < database/schema.sql`

2. **Запуск проекта**:
   - Открыть в браузере: `http://localhost`

3. **Вход администратора**:
   - URL: `http://localhost/login.php`
   - Логин: `root`
   - Пароль: `toor` (рекомендуется сменить).

Эти команды помогут настроить и запустить проект.
